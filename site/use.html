Six elements, six classes of the same name, one layout class on the html element and, voila, you're done!

Here are the elements you need to remember. Header, footer, main, nav, and aside. Grail's boilerplate looks looks like this. 

<html class="layout-standard">
<body class="body">
    <header class="header"></header>
    <nav class="nav"></nav>
    <main class="main"></main>
    <aside class="aside"></aside>
    <footer class="footer"></footer>
</body>
</html>
